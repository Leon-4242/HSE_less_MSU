tasks-graph-1D.pdf - список алгоритмов
graph_1d_list.txt - распределение, кому какие алгоритмы делать (каждому по 2)
require-graph-1D.pdf - требования к программам
bogachev_metody_priblizheniya_funktsiy_2002.djvu - книга "Богачёв К.Ю. Методы приближения функций"
graph_qt5.tar.gz - архив с примером программы на Qt5

Инструкция по сборке и запуску:

tar -xpvf graph_qt5.tar.gz  # распаковать архив
ls                          # посмотреть содержимое каталога, должен появиться каталог graph_qt5
cd graph_qt5                # зайти в появившийся каталог
qmake-qt5                   # сгенерировать Makefile по .pro файлу
make                        # собрать проект
./graph                     # запустить собранную программу
